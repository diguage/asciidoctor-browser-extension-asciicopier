== 图标创意灵感

外面使用正十七边形包裹，中间使用两个 AsciiDoctor 图标重叠，暗示复制的意思。

正十七边形是为了纪念高斯。历史上，最伟大的数学家。

AsciiDoctor 的图标使用色号 #e40145，这也是 AsciiDoctor 图标的底色。
